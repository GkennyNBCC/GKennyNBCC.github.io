This is the Tracker information website

This site serves as the central hub for all information related to the Tracker system. Whether you're looking for updates, instructions, or troubleshooting help, everything you need can be found here. The content is organized into clear sections to help you quickly navigate and find answers.

Patch Notes

Stay informed about the latest changes to the tracker system. This section includes: New features and enhancements, Bug fixes and improvements, UI/UX updates, Behind‑the‑scenes technical adjustments

Guidance

Get step‑by‑step instructions on how to use the tracker effectively. This area includes: User walkthroughs, Screenshots and examples, Best practices for data entry, Explanations of fields, dropdowns, and workflows. If you're new to the tracker or training new staff, start here.

FAQ

Find quick answers to the questions users ask most often, such as: Common errors and how to fix them, How to add new partners or interactions, Why certain fields work the way they do, Permissions and access info. This section helps you troubleshoot issues without needing to contact support.
